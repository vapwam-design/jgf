# GitHub User Finder

## Автор
Канищев Максим

## Описание
Это приложение позволяет искать пользователей GitHub по имени через API, отображать их информацию и сохранять избранных пользователей в файл.

## Как использовать
1. Установите зависимости:
   pip install -r requirements.txt
   2. Запустите `main.py`.
3. Введите имя пользователя GitHub и нажмите "Найти".
4. Для добавления в избранное — нажмите "Добавить в избранное".
5. Список избранных отображается справа.

## Примеры
- Ввод: `torvalds`
- Ввод: `octocat`
